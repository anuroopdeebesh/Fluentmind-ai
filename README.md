# Fluentmind-ai
FluentMind is an AI-powered spoken English assistant that helps users improve their speaking skills through topic-based practice, real-time voice feedback, and intelligent scoring.
# FluentMind

AI-based Spoken English Assistant

## Setup Instructions

1. Clone the repo:
git clone <your-repo-link>

2. Install dependencies:
npm install

3. Run the project:
npm run dev
